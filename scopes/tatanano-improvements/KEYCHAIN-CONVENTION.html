🔐 TataNano Keychain Naming Convention

Standard naming for all secrets stored in macOS Passwords app (iCloud Keychain)

📅 March 2026 · v1.0

📐 The Formula

tatanano.{service}.{type}

Always lowercase · dots as separators · no spaces · account field always tatanano

💡
In the Passwords app: use the Service name field for the full tatanano.* key, and always set Account to tatanano. The password field holds the actual secret value.

📱 How to Add in Passwords App

On Mac:

1
Open Passwords app (⌘Space → "Passwords")
2
Click + to add new entry
3
Set Website = the service name (e.g. tatanano.anthropic.oauth-token)
4
Set Username = tatanano
5
Set Password = your actual secret value

On iPhone:

1
Open Settings → Passwords
2
Tap + → New Password
3
Same fields: Website = service name, Username = tatanano, Password = secret
4
Save → syncs to Mac in ~30-90 seconds via iCloud ☁️

📴 Works Offline?

✅
Yes, fully offline! iCloud Keychain stores a local encrypted copy on every device. Internet is only needed for syncing between devices. TataNano reads secrets from the local copy — no internet required.
